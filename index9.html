CSS

You don't need to include the default stylesheet, everything should be already provided by the default bootstrap stylesheet framework's module. If you want to get ride-off bootstrap, you can use still the following one as an alternative :

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/peter-power-594/ResponsiveSlides.js@2.2.0/dist/css/bootstrap-carousel.min.css">

JS

<script src="https://cdn.jsdelivr.net/gh/peter-power-594/ResponsiveSlides.js@2.2.0/dist/js/responsiveslides.min.js"></script>
<script>
document.addEventListener('DOMContentLoaded', function() {
  // Slideshow 9
  new responsiveSlides('#carouselFadeExample .carousel-inner', {
    boostrap: true
  });
});
</script>

Add the setting boostrap to true to enable the boostrap mode. The script will automatically look for existing pager or navigation
